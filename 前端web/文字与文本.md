过渡与动画

```html

<!DOCTYPE html>
<html>

<head>
    <title>WEB</title>
<style type="text/css">
    /*transition过渡
    transition-property:多属性名逗号隔开;
    transition-duration:持续时间s;
    transition-timing-function:速度;   linear           匀速
                                       ease            慢快慢
                                       ease-in          慢快
                                       ease-out         快慢
                                       ease-in-out     慢快慢*/
    #div1{
        height:30px;
        width:100px;
        line-height:30px;
        border-radius:5px;
        color:#000;
        background-color:silver;
        transition:all 1s linear;
    }
    #div1:hover{
        color:white;
        background-color:#45B823;
    }
    @keyframes mycolor{
        0%   {background-color:red;}
        30%  {background-color:blue;}
        60%  {background-color:yellow;}
        100% {background-color:green;}
    }
    #div2{
        height:30px;
        width:100px;
        line-height:30px;
        border-radius:5px;
        color:#000;
        background-color:silver;
    }
    #div2:hover{
        animation:mycolor 5s linear;
    }
</style>
</head>
<body>
   <div id=div1>Web Design</div>
   <div id=div2>web</div>
</body>
</html>

```
